Services & Communication

Simaju Core menyediakan layanan komunikasi multi-channel terintegrasi (Email, WhatsApp, Telegram).

Email Service

communication.service.js
await CommunicationService.sendEmail(
  'user@example.com', 
  'Hello!', 
  'This is a test email'
);

WhatsApp Integration

Anda dapat mengirim pesan WhatsApp melalui gateway terdaftar dengan sangat mudah.

JAVASCRIPT
await CommunicationService.sendWhatsApp(
  '628123456789', 
  'Halo, ini notifikasi dari Simaju!'
);

Custom Middlewares

Anda dapat membuat middleware kustom untuk berbagai kebutuhan seperti logging audit, transformasi data, dsb.

logger.middleware.js
module.exports = (req, res, next) => {
  console.log(`[${new Date().toISOString()}] ${req.method} ${req.url}`);
  next();
};

Logging & Debug

Simaju Core menyertakan logger tingkat industri yang mendukung level log (info, warn, error) dan rotasi file secara otomatis.

JAVASCRIPT
const logger = require('./src/utils/logger');

logger.info('User logged in successfully');
logger.error({ err }, 'Database connection failed');
