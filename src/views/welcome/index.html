src/controllers/user.controller.ts
import { Controller, Get, UseAuth } from '@simaju/core';
import { UserService } from '../services/user.service';

@Controller('/api/users')
export class UserController {
  constructor(private userService: UserService) {}

  // Mengambil data pengguna yang sedang login
  @Get('/me')
  @UseAuth()
  async getProfile(req) {
    return await this.userService.findById(req.user.id);
  }
}

Mengapa Memilih Simaju?

Dibangun dari pengalaman nyata dalam menangani aplikasi skala besar, Simaju menyediakan semua yang Anda butuhkan secara langsung (Out-of-the-box).

database

Multi-DB Migration

Sistem migrasi otomatis yang mendukung MySQL, PostgreSQL, SQLite, dan MongoDB secara native dengan perintah CLI.

bug_report

Pretty Error Page

Pengalaman debugging premium dengan tampilan error bergaya Laravel Ignition yang menunjukkan snippet kode bermasalah.

domain

SaaS & Multi-Tenant

Middleware identifikasi tenant dan autentikasi API Key sudah tertanam untuk membangun aplikasi SaaS skala besar.

database

Database Agnostic

Gunakan PostgreSQL, MySQL, atau MongoDB. Simaju memiliki lapisan abstraksi data yang membuat perpindahan database menjadi mulus.

code_blocks

First-Class TypeScript

Dukungan tipe data end-to-end yang ketat. Nikmati pengalaman autocompletion yang sempurna di editor favorit Anda.

school

LMS & E-Learning

Sistem manajemen pembelajaran lengkap dengan Course Builder, Kuis Interaktif, dan Sertifikat Otomatis.
