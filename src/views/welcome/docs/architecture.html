Arsitektur

Sistem Modul

Simaju Core menganut filosofi Modular Architecture di mana setiap fitur besar aplikasi dipisahkan ke dalam modul mandiri di dalam direktori src/modules/.

Struktur Modul Standar

Setiap modul biasanya terdiri dari file-file berikut:

  • module.route.js: Definisi endpoint HTTP untuk modul ini.
  • module.controller.js: Penanganan request dan response.
  • module.service.js: Logika bisnis utama dan interaksi database.
  • module.validation.js: Aturan validasi input (menggunakan Joi atau Validator).
  • module.model.js: Abstraksi ORM untuk entitas modul ini.
verified

Keuntungan: Dengan pola ini, tim pengembang dapat bekerja pada modul yang berbeda tanpa risiko konflik kode yang tinggi.

Arsitektur Inti (Core)

Jantung dari Simaju Core adalah sistem bootloader yang sangat efisien yang mengatur inisialisasi aplikasi.

Lifecycle Request

  1. Booting: index.js memanggil App.boot().
  2. Config Loading: Memuat konfigurasi dari src/config.
  3. DB Init: Menyiapkan pool koneksi database.
  4. Middleware Registration: Memasang middleware global.
  5. Module Loading: Mendaftarkan semua route dari src/core/router.js.
