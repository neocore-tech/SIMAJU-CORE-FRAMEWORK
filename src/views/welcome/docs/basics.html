Dasar (Basics)

Routing

Routing di Simaju Core berbasis pada Express Router. Setiap modul mendaftarkan rutenya sendiri yang kemudian dikumpulkan oleh router global.

product.route.js
const express = require('express');
const router = express.Router();
const ProductController = require('./product.controller');

router.get('/', ProductController.getAll);
router.get('/:id', ProductController.getById);

module.exports = router;

Controllers

Controller adalah penghubung antara Route dan Service. Gunakan controller untuk melakukan validasi input dan memformat response.

product.controller.js
class ProductController {
  async getAll(req, res) {
    const products = await ProductService.findAll();
    return res.json({ status: true, data: products });
  }
}
module.exports = new ProductController();

Validation

Gunakan middleware validasi untuk memastikan data yang masuk sesuai dengan kontrak API Anda.

product.validation.js
const UserValidation = {
  create: (req, res, next) => {
    if (!req.body.name) return res.status(400).json({ error: 'Name is required' });
    next();
  }
};
