Sistem Modul Internal

Modul adalah tulang punggung dari aplikasi Simaju. Berbeda dengan Plugin yang bersifat dinamis, Modul digunakan untuk membangun fitur inti aplikasi yang terintegrasi erat dengan core system.

🚀 Automasi dengan CLI

Gunakan perintah berikut untuk membuat modul baru dengan struktur premium dalam hitungan detik:

./simaju make:module <nama_modul>

Struktur File Modul

Setiap modul di src/modules/ mengikuti pola arsitektur **Controller-Service-Route**:

  • Controller: Menangani request HTTP dan mengirim response.
  • Service: Berisi logika bisnis utama dan interaksi database.
  • Route: Mendefinisikan endpoint API untuk modul tersebut.
  • Validation: Skema validasi input (Joi/Zod).

Langkah Membuat Modul

1. Jalankan perintah generator:

./simaju make:module product

2. Daftarkan rute di src/core/router.js:

router.use('/products', require('../modules/product/product.route'));
