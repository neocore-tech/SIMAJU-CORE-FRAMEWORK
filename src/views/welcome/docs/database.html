Database

Simaju ORM

Simaju Core menyertakan ORM berbasis Active Record yang sangat intuitif. Ini memungkinkan Anda bekerja dengan database menggunakan objek JavaScript murni.

product.model.js
const Model = require('../../database/model');

class Product extends Model {
  static table = 'products';
}

// Penggunaan di Service
const item = await Product.find(101);
item.stock += 10;
await item.save();

Migrasi & Seeding

Gunakan sistem migrasi untuk mengelola perubahan skema database secara tim dan terkontrol.

Membuat Migrasi

BASH
./mji make:migration create_products_table

Menjalankan Migrasi

BASH
./mji run migrate

Query Builder

Jika Anda butuh query yang lebih kompleks, gunakan Query Builder yang chainable.

JAVASCRIPT
const DB = require('./src/database');

const users = await DB.table('users')
  .select('id', 'name', 'email')
  .where('status', 'active')
  .orderBy('created_at', 'desc')
  .limit(10)
  .get();
