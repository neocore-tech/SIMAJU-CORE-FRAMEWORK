Memulai Simaju Core

Selamat datang di dokumentasi resmi Simaju Core Framework. Framework ini dirancang untuk mempercepat pengembangan backend Node.js dengan struktur yang robust, modular, dan siap untuk skala produksi.

lightbulb

Tips: Jika Anda terbiasa dengan Laravel atau NestJS, Anda akan merasa sangat familiar dengan konsep modular dan arsitektur yang kami gunakan.

Prasyarat

Sebelum memulai, pastikan lingkungan pengembangan Anda memenuhi persyaratan berikut:

  • Node.js: Versi 18.x (LTS) atau lebih tinggi.
  • NPM: Versi 9.x atau lebih tinggi.
  • Database: PostgreSQL (Direkomendasikan), MySQL, atau SQLite untuk lokal.

Instalasi Cepat

Gunakan langkah-langkah berikut untuk menginisiasi proyek Simaju Core di mesin lokal Anda:

BASH
# Klon direktori proyek
cd framework-simaju-core

# Instalasi semua dependensi
npm install

# Setup environment (Salin dari example)
cp .env.example .env

# Jalankan migrasi database
./mji run migrate

# Mulai server dalam mode development
./mji run dev

Struktur Direktori

Simaju Core mengikuti pola Modular Domain Driven Design. Berikut adalah penjelasan fungsi setiap direktori utama:

folder src/core

Berisi bootloader aplikasi, routing global, dan logika inti framework yang tidak boleh diubah secara sembarangan.

folder src/modules

Rumah bagi semua fitur aplikasi Anda. Setiap folder di sini adalah "Module" mandiri (seperti Auth, User, Product).

folder src/database

Berisi driver database, manager migrasi, seeder, dan definisi model ORM.

folder src/middlewares

Penyaring request global seperti autentikasi, logging, dan penanganan error (Pretty Error).

folder src/config

Semua parameter konfigurasi aplikasi tersentralisasi di sini (DB, App, Auth, dsb).
