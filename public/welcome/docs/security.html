Keamanan

Otentikasi (Auth)

Simaju Core menyediakan sistem otentikasi berbasis API Key dan JWT. Anda dapat mengaktifkannya melalui middleware pada route tertentu.

Menggunakan Middleware Auth

user.route.js
const auth = require('../../middlewares/auth.middleware');

// Semua rute di bawah ini memerlukan validasi
router.use(auth);

router.get('/me', UserController.profile);

Otorisasi (RBAC)

Gunakan sistem Role-Based Access Control untuk membatasi aksi berdasarkan tingkat akses pengguna.

JAVASCRIPT
const authorize = (roles) => (req, res, next) => {
  if (!roles.includes(req.user.role)) {
    return res.status(403).json({ error: 'Forbidden' });
  }
  next();
};

router.delete('/:id', authorize(['admin']), ProductController.delete);

Keamanan API

Simaju Core secara otomatis menyertakan Helmet.js dan perlindungan dasar terhadap CSRF serta pembatasan Rate Limiting untuk mencegah serangan Brute Force.

warning

Penting: Selalu gunakan HTTPS di lingkungan produksi dan simpan JWT_SECRET Anda di dalam file .env yang aman.
